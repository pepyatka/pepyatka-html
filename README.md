HTML client for Pepyatka
========================

Setting up your environment
---------------------------

1. Install nodejs
2. Run `npm install`
3. Run `bower install`
4. Run `gulp watch` and edit scss files.

Running the application
-----------------------

1. `node index.js`
2. Open [http://localhost:3333/](http://localhost:3333/)
